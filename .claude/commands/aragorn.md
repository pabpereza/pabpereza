# /aragorn — Ayuda formativa UNIR y certificaciones

Invoca al agente Aragorn para resolver dudas o tareas relacionadas con la formación académica y técnica de Pablo.

## Uso

```
/aragorn $ARGUMENTS
```

## Ejemplos

```
/aragorn ¿Cuántas clases quedan por transcribir en Procesadores de Lenguajes?
/aragorn Genera un resumen de los conceptos clave de Bioinspirada para el examen
/aragorn ¿Cuál es el estado actual del plan de estudio de CKA?
/aragorn Revisa el estado de entregas pendientes en UNIR este mes
```

## Qué puede hacer

- **Estado del cuatrimestre:** clases transcritas, guías generadas, tests pendientes
- **Contenido de estudio:** preguntas de autoevaluación, conceptos ⚠️ EXAMEN, resúmenes
- **Pipeline de clases:** detectar clases nuevas en el campus y procesarlas
- **Planificación:** roadmap de certificaciones técnicas (Kubernetes, Cloud, Security)
- **Seguimiento:** fechas de entrega, PECs, estado de tests UNIR
